## 3. Service Description

### 3.1 Overview

CrowByte Terminal is a desktop application built on the Electron framework for offensive security operations. The Software provides an integrated command center for penetration testing, vulnerability assessment, red team operations, and bug bounty hunting.

### 3.2 Core Capabilities

The Software includes, but is not limited to, the following capabilities:

- **Network Scanning and Reconnaissance** — Integration with industry-standard tools (including Nmap) for host discovery, port scanning, service enumeration, and OS fingerprinting.
- **Vulnerability Assessment** — Automated vulnerability scanning powered by Nuclei, with configurable templates for identifying misconfigurations, known CVEs, and security exposures.
- **SQL Injection Testing** — Integration with SQLMap for automated detection and exploitation of SQL injection vulnerabilities in authorized targets.
- **CVE Intelligence** — A searchable CVE database with severity grouping, CVSS scoring, exploit status tracking, and integration with the National Vulnerability Database (NVD).
- **AI-Powered Analysis** — Integrated AI capabilities for automated analysis, report generation, payload crafting, and intelligent security assessments. AI features are powered by multiple providers, including Anthropic's Claude and additional large language models.
- **Remote Agent Orchestration** — Fleet management and coordination of remote VPS-based security agents for distributed reconnaissance and analysis operations.
- **Terminal Emulation** — Built-in terminal with full shell access, tmux integration, and support for standard penetration testing workflows.
- **Red Team Operations** — Planning, execution tracking, and findings management for red team engagements.
- **Knowledge Management** — A structured knowledge base for storing research findings, methodologies, and security intelligence.
- **Endpoint and Fleet Management** — Monitoring and management of endpoints, VPS agents, and distributed security infrastructure.
- **Custom Agent Builder** — Tools for creating, configuring, and deploying custom AI-powered security agents with defined capabilities and instructions.

### 3.3 Desktop Application

CrowByte Terminal is a locally-installed desktop application, not a software-as-a-service (SaaS) offering. The Software is installed and executed on your local machine. Certain features require network connectivity and rely on cloud-hosted backend services as described in Section 8.

### 3.4 Feature Availability

Feature availability varies by subscription tier. CrowByte reserves the right to modify, add, or remove features at any time, with reasonable notice for material changes to paid features. Feature deprecation notices will be provided at least thirty (30) days in advance for Professional, Team, and Enterprise subscribers.

### 3.5 System Requirements
